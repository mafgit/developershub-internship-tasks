# Empathetic DistilGPT2

> This app features a fine-tuned version of `DistilGPT2` on `Estwld/empathetic_dialogues_llm` dataset.

### Note:
 Since it is a very small model and the dataset is not enough, it is still extremely limited, although differences are clearly visible.

### What I got to do:
- Formatting conversations
- Tokenization
- Masking tokens other than the assistant responses
- Special tokens, padding, end of sequence tokens, adding additional tokens
- HuggingFace, transformers
- Streamlit UI

[Visit HuggingFace Repository](https://huggingface.co/mafgit/empathetic-distilgpt2)
